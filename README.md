## Mental arithmetic math train app
***
The app is created for mental math skill training
***
### App functionality:
- Train mental arithmetic skill
- Track your progress
- Various difficulty levels

## Build Setup
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
